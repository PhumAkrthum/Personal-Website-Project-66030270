ฉันมีความตั้งใจในการเขียน HTML ที่สะอาดและถูกต้องตามหลักความหมาย (Semantic HTML) และตกแต่งด้วยเทคนิค CSS สมัยใหม่ เช่น Flexbox และ Grid
